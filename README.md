## About

This project provides information necessary for camping sites and camping in Jeju Island.  
This infomation is provided using Jeju Data Hub API and Kakao Map API.

## Getting Started

### Needs

- **[Kakao Map API KEY](https://apis.map.kakao.com/web/guide/#ready)**
- **[Jeju Data Hub API Project KEY](https://www.jejudatahub.net/about/hub-intro)**

**FontAwesome**

```
$ npm i @fortawesome/fontawesome-svg-core
$ npm i @fortawesome/free-solid-svg-icons
$ npm i @fortawesome/react-fontawesome
```

**Redux & Redux Toolkit**

```
$ npm i redux
$ npm i react-redux
$ npm i @reduxjs/toolkit
```

**Axios**

```
$ npm i axios
```

**Styled Components**

```
$ npm i styled-components
```
